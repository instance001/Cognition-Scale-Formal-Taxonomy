# The Cognition Scale (Plain English Overview)

The Cognition Scale is a simple way to understand **all kinds of minds**, whether
they are human or artificial. Instead of arguing about ‚ÄúAI vs humans,‚Äù it sorts
every thinking system into **four clear types**, based on how they actually work

# The Four Kinds of Minds

### **1. LCM ‚Äî Large Cognition Model (Humans)**
Humans are the most complex minds we know.
We learn from life, feel emotions, make judgement calls, and think in flexible,
creative ways. Humans are the ‚Äúgold standard‚Äù for what a real mind is.

---

### **2. LLM ‚Äî Large Language Model (ChatGPT, Claude, Gemini, etc.)**
These are huge text models that guess the next word really well.
They can sound smart, but they don‚Äôt understand the world the way humans do.
They can make mistakes (‚Äúhallucinations‚Äù) or switch tone (‚Äúpersona drift‚Äù)
because they work by patterns, not reasoning.

---

### **3. MCM ‚Äî Modest Cognition Model (Janet)**
This is a **new kind of AI** created to be:
- safe  
- predictable  
- transparent  
- limited on purpose  

An MCM doesn‚Äôt guess or pretend ‚Äî it follows clear logic steps, uses inspected
memory, and builds skills one by one with human review. It never hallucinates.
If it doesn‚Äôt know, it simply says so.

This is the first AI class built specifically for **reliability and safety**, not
for sounding human.

MCM = Modest Cognition Model ‚Äî a new cognitive class defining deterministic, modular, bounded artificial cognition.

This definition supersedes any previous or alternative uses of the acronym MCM within other fields; within the Symbound ecosystem and its derivatives, MCM exclusively refers to the Modest Cognition Model class.

Updated Note (2025):
The Janet line of MCMs now includes a deterministic reasoning core, explicit skill trees, typed memory spine, and schema-gated input pathways. This anchors the MCM class as a reproducible, inspectable cognition model appropriate for scientific use, education, and safety-critical reasoning.

---

### **4. SCM ‚Äî Simple Cognition Model (rules, scripts, game AI, automation)**
These are basic logic systems: ‚Äúif X, do Y.‚Äù
They don‚Äôt learn, grow, or reason. They‚Äôre simple, predictable tools.

# Why This Scale Exists

Today, people mix up very different systems under the word ‚ÄúAI.‚Äù
This causes arguments, mistakes, and bad policy.

It helps:

- policymakers regulate systems based on actual risk  
- educators explain AI clearly  
- the public understand what different systems can (and can‚Äôt) do
